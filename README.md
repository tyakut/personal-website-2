# Personal Website

A modern, responsive personal website built with HTML, CSS, and JavaScript.

## Project Structure

```
personal-website/
├── index.html          # Main HTML file
├── css/
│   └── styles.css      # Stylesheet
├── js/
│   └── script.js       # JavaScript functionality
├── assets/
│   └── images/         # Image assets folder
└── README.md           # This file
```

## Features

- **Responsive Design**: Works on all devices (desktop, tablet, mobile)
- **Smooth Scrolling**: Navigation with smooth scroll effects
- **Modern UI**: Clean and professional design
- **Interactive Elements**: Hover effects and animations
- **Contact Form**: Ready-to-use contact form (frontend only)

## Getting Started

1. Open `index.html` in your web browser
2. Customize the content in `index.html` with your personal information
3. Update styles in `css/styles.css` to match your brand
4. Add your images to `assets/images/`
5. Modify `js/script.js` to add any additional functionality

## Customization

### Colors
Edit the CSS variables in `css/styles.css`:
```css
:root {
    --primary-color: #6366f1;
    --secondary-color: #8b5cf6;
    /* ... */
}
```

### Content
- Update the navigation links in `index.html`
- Replace placeholder text with your own content
- Add your projects to the projects section
- Update the contact form as needed
